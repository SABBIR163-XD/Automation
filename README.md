## ✨ Features at a Glance

| | |
|---|---|
| ✅ **Account Check + Forget OTP** | 🔄 **Multi-threading (up to 50 threads)** |
| ⚡ **Bulk Forget (Direct OTP Send)** | 🌐 **Proxy Checker (Paid/Free)** |
| 🛡️ **Approval System (License Verify)** | 📱 **Termux Optimized** |
| 🔁 **User-Agent Rotation** | 📞 **Telegram Support** |
| 📁 **Multiple File Encoding Support** | 🚀 **Fast & Lightweight** |

## 📥 Install on (termux)

```bash
git clone https://github.com/SABBIR163-XD/Automation.git
cd Automation
chmod +x FRGT
./FRGT
```
